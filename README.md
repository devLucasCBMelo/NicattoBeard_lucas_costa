# Desafio Técnico Nicatto

O desafio tem como objetivo criar uma aplicação para uma barbearia gerar seus atendimentos.

O usuário:

- Deve poder se cadastrar na aplicação.
- Agendar seus atendimentos (Selecionar barbeiro, especialidade, dia da semana e horário).
- Ver os agendamentos marcados.
- Poder cancelar seus agendamentos.

O Admin:

- Deve poder cadastrar, atualizar e deletar barbeiros.
- Deve poder cadastrar, atualizar, deletar e vincular especialidades à um barbeiro.

<details>
  <summary><strong>🐳 Rodando com o Docker</strong></summary>
  Na raiz do projeto utilize o comando no terminal:

```
docker compose up --build
```

A API ficará disponível em:

http://localhost:3000

e o front-end ficará disponível em:

http://localhost:5173

</details>

<details>
  <summary><strong>📍 Rodando localmente</strong></summary>
  Na raiz do projeto utilize o comando no terminal:

Dentro do back-end rode o comando:

```
npm run dev
```

A API ficará disponível em:

http://localhost:3000

Dentro do front-end rode o comando

```
npm run dev
```

e o front-end ficará disponível em:

http://localhost:5173

</details>
